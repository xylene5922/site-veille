Bloquer le protocole UTIQ

Pour faire face au blocage des cookies tiers dans les navigateurs, les marketeux d’AdTech ont inventé un nouveau protocole, Utiq, 100% compatible avec le RGPD Il s’agit d’une collecte d’adresses IP et des informations sur le navigateur au travers d’un échange d’identifiants chiffrés s’appuyant sur trois jetons – ou Pass – centralisés sur une plate-forme cloud.

Ce système de tracking et de pillage de vos données de navigation est aujourd’hui utilisé, entre autres, sur tous les sites des groupe NRJ et Reworld Media, par l’Express, Orange et Intermarché.

Les trois Pass

Le ConsentPass fournit l’adresse IP ou votre numéro de mobile.
Le MartechPass donne les sites sur lesquels vous naviguez.
Le AdtechPass permet de savoir si une publicité a provoqué un achat.

Comment bloquer le protocole Utiq à partir de votre navigateur ?

Vous pouvez utiliser uBlock Origin et la liste de filtres EasyPrivacy. Seuls les navigateurs Firefox et Brave permettent d’utiliser cette liste. Vous pouvez aussi bloquer les noms de domaines préfixés par le mot utiq ou bien le script JavaScript utiqLoader.js. Ces méthodes sont en fait peu fiables du fait qu’elles reposent sur les noms des domaines ou du script utilisé.

Je n’ai pas regardé dans le détail le fonctionnement de l’extension Firefox Utiq Bloker.

L’autre solution beaucoup plus fiable est de créer les règles das uBlock Origin en bloquant :

  • l’objet Javascript window.Utiq.API ;
  • la méthode Utiq.getIdGraphEntry() ;
  • les variables mtid et atid ;
  • le cookie utiq_logging_level.

Les règles uBlock Origin
